Map & Graph search layouts

Compare six ways to add multiple objects without leaving the canvas. Each link opens the same demo with a different add-to-map / add-to-graph experience. Open Map or Graph, then try adding several objects in one session.

    Modal (current)

    Full-screen dialog. Closes and clears search after each add.

    Expandable strip

    Search bar and results drop down from the top of the canvas.

    Fixed left panel for search and results while the canvas stays visible.

    Search list and map/graph permanently side by side.
